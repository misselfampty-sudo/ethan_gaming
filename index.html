SPACE INVADERS MINI

Welcome to your arcade console! Use the arrow keys to fly your ship.

💡 How This Game Works (Code Lessons)

  • The Game Loop: We use a special function called requestAnimationFrame. It acts like a fast heartbeat, drawing the screen over and over again 60 times every second!
  • Coordinates: The Canvas is a grid. (X: 0, Y: 0) is the top-left corner. X goes right, and Y goes down. Our ship is at the bottom (high Y value).
  • Input Handlers: When you press ◀ Left Arrow or ▶ Right Arrow, the browser triggers an event. We save this state in a JavaScript object to move the ship smoothly inside the Game Loop.
